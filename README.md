FirstCompose_GnanendraS

An Android application built using **Kotlin** and **Jetpack Compose** to explore modern Android UI development with declarative design principles.
This project demonstrates clean UI structure, Compose components, and Android app fundamentals.


Features

✅ Modern UI built with Jetpack Compose
✅ Clean and responsive layout
✅ Material Design theming
✅ Beginner-friendly project structure
✅ Kotlin-based implementation

 Tech Stack

Language: Kotlin
UI Toolkit: Jetpack Compose
IDE:** Android Studio
Build System: Gradle

---

 Project Structure

```
app/
 ┣ src/main/java/... (Application source code)
 ┣ ui/theme/ (Theme, colors, typography)
 ┣ res/ (Resources)
 ┗ AndroidManifest.xml
```



 How to Run the Project

1. Clone the repository

git clone https://github.com/Gnani7s/Android-Studio-project-folder.git

2. Open in Android Studio
3. Allow Gradle Sync to finish
4. Run the app on emulator or physical device
